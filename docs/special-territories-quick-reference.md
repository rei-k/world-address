## 📋 チェックリスト / Checklist for International Shipping

### 配送前確認事項 / Pre-Shipping Verification

- [ ] 目的地がISO 3166-1に登録されているか確認
- [ ] 独自の関税システムを持つか確認
- [ ] 必要な通関書類を準備
- [ ] 独自の郵便コードフォーマットを確認
- [ ] 配送除外品目を確認

### 主要な注意点 / Key Considerations

#### 🇭🇰 香港 (HK) / Hong Kong
- 中国本土とは**別の関税領域**
- HSコード体系は独自
- 通関書類は中国とは別に必要
- 原産地証明が必要な場合あり

#### 🇲🇴 マカオ (MO) / Macao
- 中国本土とは**別の関税領域**
- カジノ関連規制に注意
- ポルトガル語・中国語併記推奨

#### 🇹🇼 台湾 (TW) / Taiwan
- 中国本土とは**完全に別の関税システム**
- "Chinese Taipei"表記がWTO公式名
- 独自の食品・医薬品規制

#### 🌍 グリーンランド (GL) / Greenland
- **EU非加盟**(デンマークは加盟)
- 漁業製品に特別規制
- 極寒地配送の特別対応必要

#### 🏝️ フェロー諸島 (FO) / Faroe Islands
- **EU非加盟**(デンマークは加盟)
- 漁業製品に特別規制
- デンマークとは別の郵便システム

#### 🏖️ カナリア諸島 (ES-CN) / Canary Islands
- EU加盟だがVATではなく**IGIC**(間接一般税)適用
- スペイン本土とは税率が異なる
- 特別な通関手続きが必要

#### 🏛️ セウタ・メリリャ (ES-CE/ML) / Ceuta & Melilla
- EU加盟だが**関税領域外**
- スペイン本土への輸送でも通関必要
- モロッコとの特別通関協定あり

#### 🏝️ プエルトリコ (PR) / Puerto Rico
- 米国領土だが**独自の税制**
- USPS配送システム利用
- 米国本土とは異なる消費税

#### 🏦 ケイマン諸島 (KY) / Cayman Islands
- 国際金融センター
- 独自通貨 KYD
- 金融サービス関連規制に注意
