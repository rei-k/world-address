## Design Principles

### Core UX Principles

1. **Simplicity First**
   - One-line input for senders
   - One-tap selection for recipients
   - Clear, minimal interfaces

2. **Privacy by Default**
   - Address never shown to sender
   - Encrypted data throughout
   - Clear privacy indicators

3. **Trust & Transparency**
   - Show what information is shared
   - Clear state progression
   - Honest time estimates

4. **Mobile-First**
   - Touch-optimized interfaces
   - Responsive designs
   - Wallet integration
